# AngularElementsDemo

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 6.0.0.

## Building and deploying Angular Elements to demos

```sh
npm run build:elements
```

## Running Demos

Angular

```sh
npm run start
```

React

```sh
npm run start:react
```

Vue.js
```
npm run start:vue
```
